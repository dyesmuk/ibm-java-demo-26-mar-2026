## 🛠 How to Use This Repo

Each module has its own repository. Clone the one you need:

```bash
git clone https://github.com/your-org/01-html-css
git clone https://github.com/your-org/02-core-java
# ... and so on
```

To get updates pushed by the trainer:

```bash
git pull
```

No GitHub account is needed to clone or view public repositories.

## ✅ Prerequisites

Before the training begins, make sure you have the following installed:

- [JDK 17+](https://adoptium.net/)

- [VS Code](https://code.visualstudio.com/) (for frontend modules)
- [Node.js LTS](https://nodejs.org/)
- [Git](https://git-scm.com/)
- [Docker Desktop](https://www.docker.com/products/docker-desktop/)
- [Postman](https://www.postman.com/)
- [Maven](https://maven.apache.org/) (or use the bundled version in your IDE)
